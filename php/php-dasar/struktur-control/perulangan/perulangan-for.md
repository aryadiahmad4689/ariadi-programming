# Perulangan For

Perulangan `for` digunakan ketika Kalian tahu sebelumnya berapa kali blok kode harus dijalankan. Struktur dasarnya melibatkan inisialisasi variabel, kondisi, dan increment/decrement variabel.

```php
<?php
for ($i = 1; $i <= 5; $i++) {
    echo $i . " ";
}
// Output: 1 2 3 4 5
?>

for ($i = 5; $i <= 1; $i--) {
    echo $i . " ";
}
// Output: 5 4 3 2 1
?>
```
